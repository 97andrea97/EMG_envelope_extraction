This repo contains:
- "EMG_envelopes.m" is the code to compute 4 different envelopes of an EMG signal: IEMG, ARV, RMS, Zero Crossing. 
- "f_notch_sel.m" the function used to apply a notch filter to the signal. 
- "EM_EMG_SQUEEZE1.txt" dataset contains the EMG signal (during the action of squeezing an hand) in the third column.
